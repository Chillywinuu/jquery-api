:hidden

选择所有隐藏的元素。

  • 1.0 新增:hidden

元素可能出于以下几个理由而被隐藏：

  • 他们的 CSS 中 display 值为 none。
  • 他们是表单元素，带有 type="hidden" 属性。
  • 他们的高和宽被设置成 0。
  • 他们的祖辈元素被隐藏了，所以他们也没有在页面中显示。

visibility: hidden 或者 opacity: 0 的元素被认为是可见的，因为实际上，他们在页面布局中占据了一定的空间。在执行隐藏效果动画的元素也认为是可见的，直到动画结束为止。在执行显示效果动画的元素，从动画的一开始就被认为是可见的了。

从 jQuery 1.3.2 起，:hidden 的检测方式发生了变动。一个元素，只要他或者他的父元素在页面布局中不占据空间，则被认为是隐藏了。CSS 中的 visibility 属性并不影响这个选择器的判断。（所以 $(elem).css('visibility','hidden').is(':hidden') == false). 访问 jQuery 1.3.2 更新信息 来获得更多内容。

注意，有些浏览器中，包括 head, title, script, style 在内的一些元素也会被认为是隐藏的元素而被选中！

示例:

显示所有隐藏的 div 并计算 type='hidden' 的 input 元素数目。

<!DOCTYPE html>
<html>
<head>
<style>
  div { width:70px; height:40px; background:#ee77ff; margin:5px; float:left; }
  span { display:block; clear:left; color:red; }
  .starthidden { display:none; }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<span></span>
  <div></div>
  <div style="display:none;">Hider!</div>
  <div></div>

  <div class="starthidden">Hider!</div>
  <div></div>
  <form>
    <input type="hidden" />

    <input type="hidden" />
    <input type="hidden" />
  </form>
  <span>

  </span>

<script>


    // in some browsers :hidden includes head, title, script, etc... so limit to body
    $("span:first").text("Found " + $(":hidden", document.body).length + 
                         " hidden elements total.");
    $("div:hidden").show(3000);
    $("span:last").text("Found " + $("input:hidden").length + " hidden inputs."); 


</script>
</body>
</html>
