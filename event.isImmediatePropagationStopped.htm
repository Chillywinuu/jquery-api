返回值:Booleanevent.isImmediatePropagationStopped()

根据事件对象中是否调用过 event.stopImmediatePropagation() 方法来返回一个布尔值。

  • 1.3 新增event.isImmediatePropagationStopped()

这个事件方法在 DOM level 3 有介绍。

示例:

检测 event.stopImmediatePropagation() 是否被调用过。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>


  <button>click me</button>
  <div id="stop-log"></div>
  

<script>



function immediatePropStopped(e) {
  var msg = "";
  if ( e.isImmediatePropagationStopped() ) {
    msg =  "called"
  } else {
    msg = "not called";
  }
  $("#stop-log").append( "<div>" + msg + "</div>" );
}

$("button").click(function(event) {
  immediatePropStopped(event);
  event.stopImmediatePropagation();
  immediatePropStopped(event);
});  


</script>
</body>
</html>
