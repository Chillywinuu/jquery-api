返回值:booleanjQuery.isWindow(obj)

检测传入的参数是否是一个 window 对象。

  • 1.4.3 新增jQuery.isWindow(obj)

    obj (Object) 用于检测是否是 window 的对象。

jQuery 经常使用这个方法，用于检测当前操作的对象是否是一个浏览器窗口（比如是当前的窗口，或者是个框架）。

示例:

确定传入的参数是否是一个 window 对象。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

Is 'window' a window? <b></b>

<script>

$("b").append( "" + $.isWindow(window) );

</script>
</body>
</html>
