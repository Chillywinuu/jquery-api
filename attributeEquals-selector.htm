[attribute="value"]

选择含有特定属性，并且其属性值精确等于给定值的元素。

  • 1.0 新增[attribute="value"]

    attribute (String) 属性名。
    value (String) 属性值，引号是强制性的。

示例:

查找所有 name 属性值为 'newsletter' 的元素，并且修改跟在他后面的 span 元素中的文本。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<div>

    <input type="radio" name="newsletter" value="Hot Fuzz" />
    <span>name?</span>

  </div>
  <div>
    <input type="radio" name="newsletters" value="Cold Fusion" />

    <span>name?</span>
  </div>
  <div>
    <input type="radio" name="accept" value="Evil Plans" />

    <span>name?</span>
  </div>

<script>

$('input[name="newsletter"]').next().text(' is newsletter');

</script>
</body>
</html>
