:odd

选择索引值是奇数的元素。索引值从 0 开始计数。同时可以参考 even。

  • 1.0 新增:odd

特别注意，跟通常的直觉相反，索引值从 0 开始计数实际上意味着，如果使用了这个奇数索引选择器 :odd ，实际选中的是匹配集合中第二个、第四个、第六个这类元素。

示例:

查找表格中的偶数行，即匹配第二行、第四行等 (索引值是 1, 3 ,5 )。

<!DOCTYPE html>
<html>
<head>
<style>

  table {
    background:#f3f7f5;
  }
  </style>
<script src="jquery.min.js"></script>
</head>
<body>

<table border="1">
    <tr><td>Row with Index #0</td></tr>
    <tr><td>Row with Index #1</td></tr>

    <tr><td>Row with Index #2</td></tr>
    <tr><td>Row with Index #3</td></tr>
  </table>

<script>

$("tr:odd").css("background-color", "#bbbbff");

</script>
</body>
</html>
