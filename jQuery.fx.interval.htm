返回值:NumberjQuery.fx.interval

动画时两帧之间的时间间隔(单位是毫秒)。

  • 1.4.3 新增jQuery.fx.interval

这个属性可以调整动画运行时，两帧之间的时间间隔。默认值是 13 毫秒。这个值越低，在较快的浏览器中动画越流畅（比如 Chrome），但是也越消耗 CPU。

由于 jQuery 只使用一个全局的动画间隔，所以修改这个属性的时候，请务必保证没有动画正在运行。

示例:

让动画在较低的帧率下运行。

<!DOCTYPE html>
<html>
<head>
<style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
<script src="jquery.min.js"></script>
</head>
<body>

<p><input type="button" value="Run"/></p>
<div></div>

<script>


jQuery.fx.interval = 100;

$("input").click(function(){
  $("div").toggle( 3000 );
});
  

</script>
</body>
</html>
