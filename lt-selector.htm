:lt(index)

选择在匹配的集合中，索引值小于 index 参数的元素。

  • 1.0 新增:lt(index)

    index (Number) 从 0 开始计数匹配元素用的索引值。

索引值相关的选择器

索引值相关的选择器 (:eq(), :lt(), :gt(), :even, :odd)，会在先前提供的选择器筛选出的元素基础上再进行筛选。进一步筛选的依据就是这个元素在原先匹配集合中的顺序。举例来说，如果一开始通过 class 选择器 (.myclass) 选中了 4 个元素，然后这四个元素的索引值会被分配为 0 到 3 ，之后就可以用这种索引值选择器来进一步筛选了。

注意，由于 JavaScript 数组的索引值是从 0 开始计数的，所以这些选择器也是以 0 开始计数的。例如，$('.myclass:lt(1)') 就能选中文档中 class 是 myclass 的第一个元素，而不是什么都选不到。作为对比， :nth-child(n) 中的 n 是以 1 开始计数的，因为 CSS 规范里是这么设定的。

示例:

查找索引值小于 4 的 td。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<table border="1">

  <tr><td>TD #0</td><td>TD #1</td><td>TD #2</td></tr>
  <tr><td>TD #3</td><td>TD #4</td><td>TD #5</td></tr>

  <tr><td>TD #6</td><td>TD #7</td><td>TD #8</td></tr>
</table>

<script>

$("td:lt(4)").css("color", "red");

</script>
</body>
</html>
