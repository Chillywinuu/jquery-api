:submit

选择所有提交按钮，即 type 为 submit 的元素。

  • 1.0 新增:submit

:submit 选择器主要用于 button 或者 input 元素。注意，某些浏览器默认把没有设置 type 的 <button> 元素看做 type="default" ，而其他浏览器(比如IE)则并非如此。

示例:

    var submitEl = $("td :submit")
      .parent('td')
      .css({background:"yellow", border:"3px red solid"})
