返回值:jQuerypushStack(elements)

将一个DOM元素集合加入到jQuery栈。

  • 1.0 新增pushStack(elements)

    elements (Array) 将要压入jQuery栈的元素，用于生成一个新的jQuery对象。

  • 1.3 新增pushStack(elements, name, arguments)

    elements (Array) 将要压入jQuery栈的元素，用于生成一个新的jQuery对象。
    name (String) 用于生成元素数组的jQuery方法名。
    arguments (Array) 传递给jQuery方法的参数(用于序列化)。

示例:

Add some elements onto the jQuery stack, then pop back off again.

jQuery 代码:
jQuery([])
    .pushStack( document.getElementsByTagName("div") )
        .remove()
    .end();
