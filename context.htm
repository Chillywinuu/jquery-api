返回值:Elementcontext

传给jQuery()的原始的DOM节点内容，即jQuery()的第二个参数。如果没有传递过第二个参数，那么 context 指向当前的文档(document)。

  • 1.3 新增context

.live() 方法绑定事件处理函数时会用到这个属性，用于检测将要使用的元素根节点以实现事件委派。需要实现类似功能的插件就很可能需要使用这个属性。

这个属性的值通常等价于 document 。因为 jQuery 对象的 context 参数默认情况下是无需提供的。有时候这个值会有变化，比如是在 <iframe> 中搜索，或者 XML 文档中搜索时。
